## 📁 Days 1–7 Problem Details

### Day 1: [Reverse String](file:///home/dinesh/Desktop/dsa%20questions/day%201/reverse_string.py)
- **Problem**: Reverse a given string in Python.
- **Approach**: Shows both pythonic slicing (`s[::-1]`) and standard iterative loops for interview discussions.

### Day 2: [Palindrome Check](file:///home/dinesh/Desktop/dsa%20questions/day%202/palindrome.py)
- **Problem**: Check if a string is a palindrome.
- **Approach**: Covers the $O(1)$ auxiliary space two-pointer approach, comparing characters from start and end.

### Day 3: [Largest in an Array](file:///home/dinesh/Desktop/dsa%20questions/day%203/largest_in_an_array.py)
- **Problem**: Find the largest number in a list.
- **Approach**: Iterative comparison tracker and Python's built-in `max()` function.

### Day 4: [Two Sum](file:///home/dinesh/Desktop/dsa%20questions/day%204/two_sum.py)
- **Problem**: Find indices of two numbers that add up to a target.
- **Approach**: Discusses the brute-force $O(n^2)$ search and the optimal $O(n)$ Hash Map lookup.

### Day 5: [Best Time to Buy & Sell Stock](file:///home/dinesh/Desktop/dsa%20questions/day%205/best_time_to_buy_and_sell_stock.py)
- **Problem**: Find the max profit possible by buying and selling a stock (LeetCode 121).
- **Approach**: A single-pass greedy scan maintaining the minimum buying price and updating max profit. Includes edge-case handling.

### Day 6: [Remove Duplicates](file:///home/dinesh/Desktop/dsa%20questions/day%206/remove_duplicates.py)
- **Problem**: Remove duplicate elements from an array.
- **Approach**: Explores order-preserving hash set approach and the in-place two-pointer algorithm for sorted arrays (LeetCode 26).

### Day 7: [Second Largest Element](file:///home/dinesh/Desktop/dsa%20questions/day%207/second_largest_element.py)
- **Problem**: Find the second largest distinct element in an array.
- **Approach**: Demonstrates the $O(n)$ single-pass scan method keeping track of the largest and second largest elements (handles duplicates correctly).
